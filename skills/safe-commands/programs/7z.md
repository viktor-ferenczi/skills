# 7z (7-Zip)

**Platforms:** Multi-platform
**Category:** File Operations

## Safe Commands (Read-Only)

| Goal | Command |
|------|---------|
| List contents | `7z l archive.7z` |
| List verbose | `7z l -slt archive.7z` |
| Test integrity | `7z t archive.7z` |
| Extract to stdout | `7z e -so archive.7z file` |
| Show tech info | `7z i` |
| Help | `7z --help` |

## Dangerous Commands (Avoid in Safe Mode)

| Command | Risk |
|---------|------|
| `7z x archive.7z` | Extracts files (may overwrite) |
| `7z a archive.7z files` | Creates archives |
| `7z d archive.7z` | Deletes from archive |
| `7z u archive.7z` | Updates archive |
| Any 7z extraction without review | May overwrite files |

## Recommended Safe Usage

```bash
#!/bin/bash
# Safe 7z Inspection Script

# List contents
7z l archive.7z

# List verbose with tech info
7z l -slt archive.7z

# Test integrity
7z t archive.7z

# Show version info
7z i
```
