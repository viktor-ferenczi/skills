# ant

**Platforms:** Multi-platform  
**Category:** Java Build Tool

## Quick Reference

| Goal | Command |
|------|---------|
| Quiet build | `ant -q` |
| Silent build | `ant -S` |
| No interactive input | `ant -noinput` |
| Log to file | `ant -logfile build.log` |

## Command-Line Flags

- `-q` or `-quiet`: Quiet mode — reduced output
- `-S` or `-silent`: Silent mode — no output except errors
- `-noinput`: Disable interactive input (fail instead of prompting)
- `-logfile`: Redirect output to a log file
- `-emacs`: Emacs-friendly output (no adornments, useful for CI parsing)
