# bzip2 / bunzip2 / bzcat

**Platforms:** Multi-platform  
**Category:** Compression

## Quick Reference

| Goal | Command |
|------|---------|
| Quiet compress | `bzip2 -q file` |
| Force overwrite | `bzip2 -f file` |

## Command-Line Flags

- `-q` or `--quiet`: Suppress non-essential warning messages
- `-f` or `--force`: Force overwrite of output files (avoids interactive prompt)
