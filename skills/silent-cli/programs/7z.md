# 7z / 7za

**Platforms:** Multi-platform  
**Category:** Compression

## Quick Reference

| Goal | Command |
|------|---------|
| Auto-yes extract | `7z x -y archive.7z` |
| Auto-yes create | `7z a -y -r archive.7z dir/` |

## Command-Line Flags

- `-y`: Assume Yes on all queries (suppresses interactive prompts)

## Recommended Unattended Usage

```bash
7z a -y -r archive.7z dir/           # Create recursively, auto-yes
7z x -y -o/dest archive.7z           # Extract to directory
7z t -y archive.7z                   # Test archive
```
