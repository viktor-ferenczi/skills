# 7z / 7za

**Platforms:** Multi-platform  
**Category:** Compression

## Quick Reference

| Goal | Command |
|------|---------|
| Create archive | `7z a archive.7z files` |
| Extract | `7z x archive.7z` |
| List contents | `7z l archive.7z` |
| Test archive | `7z t archive.7z` |

## Command-Line Flags

```bash
7z a archive.7z files                # Add to archive
7z x archive.7z                      # Extract with full paths
7z e archive.7z                      # Extract (flat)
7z l archive.7z                      # List contents
7z t archive.7z                      # Test archive
7z d archive.7z file                 # Delete from archive
7z u archive.7z files                # Update archive
```
- `a`: Add
- `x`: Extract with full paths
- `e`: Extract (no paths)
- `l`: List
- `t`: Test
- `d`: Delete
- `u`: Update
- `-y`: Assume Yes on all queries
- `-r`: Recurse
- `-o`: Output directory
- `-p`: Password
- `-mhe=on`: Encrypt headers
- `-mx=9`: Maximum compression

## Recommended Unattended Usage

```bash
7z a -y -r archive.7z dir/           # Create recursively, auto-yes
7z x -y -o/dest archive.7z           # Extract to directory
7z t -y archive.7z                   # Test archive
```
